Major Endpoints

create_major(major: Major)

Create a new major in the database.

Parameters:

major (Major) – The major object to be created.

Returns:

The created major object.

Return type:

Major

Raises:

HTTPException – If the creation of the major fails.

delete_major(name: str)

Delete a major by its name.

Parameters:

name (str) – The name of the major to delete.

Returns:

A dictionary with a message indicating the major was deleted successfully.

Return type:

dict

Raises:

HTTPException – If the major is not found in the database.

get_all_majors()

Fetch all majors from the database.

Returns:

All majors from the database

Return type:

list[Major]

get_major(name: str)

Retrieve a major by its name.

Parameters:

name (str) – The name of the major to retrieve.

Returns:

The major object if found.

Return type:

Major

Raises:

HTTPException – If the major is not found in the database.
