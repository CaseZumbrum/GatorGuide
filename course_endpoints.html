Course Endpoints

create_course(course: Course)

Create a new course in the database.

Parameters:

course (Course) – The course object to be created.

Returns:

The created course object.

Return type:

Course

Raises:

HTTPException – If the creation of the course fails.

delete_course(code: str)

Delete a single course by its code.

Parameters:

code (str) – The course code to delete.

Returns:

A message indicating successful deletion of the course.

Return type:

dict

Raises:

HTTPException – If the course is not found in the database.

get_all_courses()

Fetch all courses from the database. (does not include prereqs)

get_corequisites(code: str)

Fetch corequisite groups for a given course.

Parameters:

code (str) – The course code.

Returns:

A dictionary with the course code and a list of corequisite groups.

Return type:

dict

Raises:

HTTPException – If the course is not found.

get_course(code: str)

Fetch a single course by its code.

Parameters:

code (str) – The course code to retrieve.

Returns:

The course object if found.

Return type:

Course

Raises:

HTTPException – If the course is not found in the database.

search_courses(query: str = Query(PydanticUndefined), limit: int = 10)

Search for courses by code or name.

Parameters:

  • query (str) – The query to search for. Must be at least 2 characters long.

  • limit (int, optional) – The maximum number of results to return. Defaults to 10.

Returns:

A list of matching courses. Each course will appear only once in the results.

Return type:

list[Course]
